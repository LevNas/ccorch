# US-006: Marketplace Distribution

## User Story

**As a** Claude Code plugin developer
**I want to** distribute ccorch via the claudecode-plugins marketplace
**So that** other users can easily install it with `/plugin install ccorch@levnas-plugins`

## Acceptance Criteria (EARS Notation)

### Functional Requirements

- **REQ-006-001**: The ccorch repository shall contain `.claude-plugin/plugin.json` conforming to the Claude Code plugin specification
- **REQ-006-002**: The claudecode-plugins `marketplace.json` shall include a ccorch entry with source `https://github.com/LevNas/ccorch.git` (ref: main)
- **REQ-006-003**: `README.md` shall document installation instructions, usage, and prerequisites

## plugin.json Structure

```json
{
  "name": "ccorch",
  "description": "tmux-based orchestration plugin for Claude Code. Provides /ccor skill for multi-level task delegation.",
  "version": "0.1.0",
  "author": "LevNas",
  "license": "MIT",
  "keywords": ["orchestration", "tmux", "parallel", "delegation"]
}
```
