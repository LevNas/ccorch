# US-005: ccmemo Integration

## User Story

**As a** Claude Code user who uses ccmemo
**I want to** discoveries and task progress from orchestration to be recorded in ccmemo
**So that** important findings persist in the knowledge base even after temporary orchestration files are cleaned up

## Acceptance Criteria (EARS Notation)

### Functional Requirements

- **REQ-005-001**: If the ccmemo plugin is installed, the Main Brain shall be able to record discoveries using the `record-knowledge` skill
- **REQ-005-002**: If the ccmemo plugin is installed, the Main Brain shall be able to track the overall orchestration plan using the `plan-task` skill
- **REQ-005-003**: If the ccmemo plugin is not installed, the system shall skip ccmemo-related features and operate correctly as a standalone plugin
- **REQ-005-004**: ccmemo availability shall be auto-detected at launch; no explicit environment variable or flag shall be required
